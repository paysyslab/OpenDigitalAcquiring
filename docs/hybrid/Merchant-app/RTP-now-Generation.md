# Raast RTP

## RTP Mode Selection

Allows merchant to choose RTP type.

- Now option available.
- Later option available.
- Merchant proceeds based on selected mode.



## Customer Account Verification

Verifies recipient before generating RTP.

- Merchant will enter customer's Raast ID or IBAN field available.
- Expiry (minute) field available.
- Disclaimer displayed:
  - Payment credited after confirmation.
  - Provide goods/services after confirmation message.
- Verify button available.

## RTP Generation

Displays RTP configuration before request generation.

- Recipient name displayed.
- Recipient IBAN displayed.
- Bank name displayed.
- Enter Amount field available.
- Numeric keypad available for amount entry.
- Expiry (minute) selection available.
- Preset expiry options available (5m, 15m, 30m, 60m, 120m, 180m).
- Expiry text field available
- Expiry time must not exceed 180m
- Optional comments field available.
- Generate Request button available.

## RTP Generated

Displays generated RTP confirmation.

- Amount displayed.
- Expiry duration displayed.
- Remaining expiry time displayed.
- OK button available.

## RTP Expired

Displays RTP expiration status.

- Amount displayed.
- Expired message displayed.
- Expiry time displayed as 00:00:00.
- Re-Generate option available.
- New RTP option available.
