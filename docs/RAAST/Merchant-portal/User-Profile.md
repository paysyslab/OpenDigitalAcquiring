# User Profile

## Screen Overview

The **User Profile** module allows the merchant to view their registered account details within the Merchant Portal.

This screen provides read-only visibility of core merchant information and associated contact details.



## Profile Header Section

The top section of the screen includes:

- Profile avatar (default or uploaded image)
- Merchant display name
- Registered email address

This section confirms the logged-in merchant identity.



## Merchant Information Section

The following fields are displayed:

### Merchant ID
- Unique identifier assigned to the merchant
- Non-editable
- Used for transaction routing and system identification

### Merchant Name
- Official registered merchant name
- Non-editable

### Doing Business As (DBA)
- Business trading name (if applicable)
- Non-editable

### Mobile Number
- Registered mobile number associated with the merchant
- Non-editable
- Used for OTPs, alerts, and communication



## Data Characteristics

- All fields are view-only
- No inline editing is available on this screen
- Changes (if required) must follow back-office approval or administrative process
