# Add MDR Profile 

## Overview
The **Add MDR Profile** functionality allows creation of a new MDR profile by defining profile name, maximum limit, MDR configuration, and applicable slab values.

## Field Details

**MDR Profile Name**
- Name assigned to the MDR profile.

**Global Maximum Limit**
- Defines the global maximum limit associated with the MDR profile.

**MDR Type**
- Dropdown field used to select the MDR type.

**MDR Value**
- Specifies the MDR value applicable to the selected type.

**Slab Low**
- Defines the lower value of the MDR slab range.

**Slab High**
- Defines the upper value of the MDR slab range.

## Additional Configuration

**Add More**
- Allows adding additional MDR slab configurations.

## Submission

**Submit Button**
- Saves the entered MDR profile details.
