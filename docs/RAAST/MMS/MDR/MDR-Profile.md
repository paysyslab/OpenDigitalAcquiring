# MDR Profile View

## Overview
The **MDR Profile View** functionality provides an interface to view configured MDR profiles within the system. It displays profile details along with maximum amount configuration and creation information.

## MDR Profile Listing Details

**Profile Name**
- Displays the name of the MDR profile.

**Global Max Amount**
- Shows the maximum global amount configured for the MDR profile.

**Created On**
- Displays the date when the MDR profile was created.

**View**
- Allows viewing details of the selected MDR profile.
