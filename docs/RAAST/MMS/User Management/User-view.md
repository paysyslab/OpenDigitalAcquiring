# View Users

## Overview
The **View Users** functionality provides a centralized interface to view all configured system users. It allows users to search, review, and perform actions on user accounts based on their assigned roles and permissions.

## User Listing Details

**Username**
- Displays the unique username assigned to each user.

**ID**
- System-generated unique identifier of the user.

**Created Date**
- Shows the date and time when the user account was created.

**Role ID**
- Displays the role identifier assigned to the user.

## Search Functionality

**Search By Username**
- Allows searching users by entering the username.
- Clicking the search button filters the user list accordingly.

## Available Actions

**View**
- Opens detailed view of the selected user.

**Update**
- Allows modification of user details.

**Disable**
- Disables the selected user account.

**Lock**
- Locks the user account.

**Reset Password**
- Initiates password reset for the selected user.
