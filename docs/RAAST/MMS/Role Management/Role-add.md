# Add / Update Role

## Overview
The **Add / Update Role** functionality allows creation and modification of roles within the system. Roles define access levels and permissions assigned to users based on their user type.

## Field Details

**Name**
- Name of the role being created or updated.

**Description Name**
- Description associated with the role.

**User Type**
- Dropdown field used to select the applicable user type for the role.

## Submission

**Submit Button**
- Saves the entered or updated role details.
- Sends the request for checker approval.

- After submission or update, the role record is routed for checker approval.
- The role becomes effective only after approval.
