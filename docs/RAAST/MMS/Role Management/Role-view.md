# View Roles

## Overview
The **View Roles** functionality provides a centralized interface to view all configured roles within the system. It allows users to search, review, and access role details along with associated permissions and user type information.

## Role Listing Details

**Role**
- Displays the name of the role.

**Description**
- Shows the description associated with the role.

**Permissions**
- Indicates the number of permissions assigned to the role.

**User Type**
- Displays the user type linked with the role.

## Search Functionality

**Search By Role Name**
- Allows searching roles by entering the role name.
- Clicking the search button filters the role list accordingly.

## Checker Approval

- After submission or update, the role record is routed for checker approval.
- The role becomes effective only after approval.
