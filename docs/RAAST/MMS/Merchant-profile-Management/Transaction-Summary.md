# Transaction Summary

## P2M Transaction Overview

The P2M transactions screen facilitates the tracking and management of transactions where customers make payments to merchants. The table includes detailed transaction data for each processed payment.

## Key Functionalities

**Transaction Overview:** Displays key details for each transaction such as:

**Payment Date:** The timestamp when the transaction was processed.

**Transaction ID:** A unique identifier for the transaction.

**Merchant Name:** The name of the merchant receiving the payment.

**MID (Merchant ID):** Unique identifier for the merchant.

**TID (Terminal ID):** The terminal identifier used for the transaction.

**Doing Business As (DBA):** The name under which the merchant is doing business.

**Payee Acc Title:** The title of the payee account.

**Payee IBAN:** International Bank Account Number for the payee.

**Sender Acc Title:** The title of the sender’s account.

**Sender Acc IBAN:** International Bank Account Number for the sender.
