# Raast QR Code

## Dynamic QR

### QR Generation

Displays dynamic QR configuration before generating the QR.

- Dynamic QR tab selected.
- Amount entry field displayed.
- Numeric keypad available for amount input.
- Expiry (minute) selection available.
- Preset expiry options available (5m, 15m, 30m, 60m, 120m, 180m).
- Expiry text field available
- Expiry time must not exceed 180m
- Generate QR Now button enabled after valid input.

### Generated QR

Displays generated dynamic QR code.

- Merchant name displayed.
- Merchant IBAN displayed.
- Amount displayed.
- QR code displayed.
- Till Code displayed.
- MID displayed.
- Expiry time displayed.
- Save in gallery option available.
- Share option available.

### Transaction Status

Displays transaction outcome after QR payment.

#### Transaction Successful

- Success popup displayed with amount.
- Transaction date displayed.
- RTP ID displayed.
- Received From details displayed.
- Received via channel displayed.
- Done button available.

<div style={{ width : '30%' }}>
  ![RAAST DQRC Status](/img/DQRC/4.png)
</div>

#### Transaction Declined

- Decline popup displayed with amount.
- Transaction date displayed.
- OK button available.
