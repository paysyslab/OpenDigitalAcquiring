# Existing Merchant Login

## Overview
This functionality allows an existing merchant to log in to the Merchant App using their registered mobile number and a 4-digit login PIN. The screen also provides options to recover the PIN and to log in using biometric authentication.

## Login Screen

### Tabs
**Login / Register**
- The merchant selects the **Login** tab to access the login inputs.

### Inputs

**Mobile Number**
- A single input field to enter the merchant’s registered mobile number.

**Enter your PIN**
- A 4-digit PIN input using separate PIN boxes.

### Actions

**LOGIN**
- Validates the entered mobile number and PIN.
- On successful validation, the merchant is logged into the app.

**Forgot PIN**
- Initiates the PIN recovery process for the entered mobile number.

**Biometric Login**
- Allows the merchant to log in using biometric authentication via the fingerprint option.
