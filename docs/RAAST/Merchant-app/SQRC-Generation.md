# Raast QR Code

## Static QR

### Without Amount

Displays a static QR code without a predefined transaction amount.

- Static QR tab selected.
- Without Amount option selected.
- Merchant name displayed.
- Merchant IBAN displayed.
- QR code displayed.
- Till Code displayed.
- Save in gallery option available.
- Share option available.

</div>

### With Amount

Displays a static QR code with a predefined transaction amount.

- Static QR tab selected.
- With Amount option selected.
- Amount entry provided.
- Generate QR Now option available.
- Merchant name displayed.
- Merchant IBAN displayed.
- Amount displayed with QR.
- QR code displayed.
- Till Code displayed.
- Save in gallery option available.
- Share option available.
